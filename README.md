# node-webpack-boilerplate
Node.js barebones webpack boilerplate... 
## Install
```
git clone https://github.com/Aboriginal-Technology-Solutions/node-webpack-boilerplate

cd node-webpack-boilerplate

yarn

yarn dev

```
